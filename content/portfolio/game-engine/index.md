As part of a class on game engines, I developed a game engine demo in C++ that included:

- SIMD math library
- DirectX 9 graphics
- Skeletal animation
- Custom shaders, including phong shading
- Camera, along with mouse and keyboard controls
- Memory allocators (stack and pool)
